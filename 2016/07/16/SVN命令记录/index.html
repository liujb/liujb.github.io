SVN命令

svn命令详解

基础

  1. svn list svn://svndomain/server/static/ – show list
  2. svn co svn://svndomain/server/trunk/api/v2/views – checkout
  3. svn add xx – add files or folders
  4. svn import $DIR $URL -m “Something your want to mark.” – import local folder to svn and commit to remote
  5. svn rm file – remove
  6. svn mv old_file new_file – move or rename
  7. svn up – update
  8. svn ci filename -m "checkin note” – checkin
  9. svn st -v/-u – show status
  10. svn log filename – show logs
  11. svn cat -r 10528 home.html – show history
  12. svn delete filename – delete
  13. svn delete –force over-there – force to delete

进阶

  1. svn export svn://svndomain/server/trunk/api/v2/views – use for release project, without .svn folder
  2. svn diff -r 12302:12304 home.html – diff two files
  3. svn diff filename – show difference from the working file and the latest version
  4. svn diff get_hb_list.tpl |vim -R - – use vim to diff two files
  5. svn diff filename -r1234:2345[|vim -R -] – use vim to diff two files
  6. svn up –r 200 file.c – 将本地的file.c还原为200版本，【本地是拿下来了，版本库端并没有改变】
  7. svn mkdir newdir
  8. svn mkdir newdir -m "Making a new dir" svn://svndomain/server/
  9. svn resolve –accept working compressed，svn revert compressed – 解决目录compressed树冲突
  10. svn revert filename
  11. ci的时候出现 xx.html is out of date – 解决方法就是先update一下，然后再ci

问题列表

冲突的解决方法：

  1. 先将.mine文件拷贝到你自己的文件
  2. 先跟test.html跟低版本的比较
  3. 再将test.html跟高版本的比较
  4. 解决完冲突之后再diff一下
  5. 然后设置svn resolved test.html
  6. 然后ci
