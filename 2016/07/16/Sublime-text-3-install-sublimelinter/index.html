Sublime-text-3-install-sublimelinter

写在前面

楼主是个喜欢折腾的同学，之前也在sublime text中配置过各类语言的linter，jshint，csslint等。来来回回折腾几回后，也没有完全用上，或者说没弄明白吧，再次来配置的时候还是不能一蹴而就，故作记录。

本文假设

  1. 你使用的sublime text 3，因为sublime text 2的sublimelinter已经不维护了。
  2. 最好使用的是unix like环境。
  3. 已具备翻墙，肉翻的人直接忽略。

环境准备

  1. sublime text 3且安装了package control，还没配置的请点击此处
  2. nodejs环境好使，因为jshint和csslint依赖node做check引擎，若不需要直接忽略吧。
  3. 很多插件都依赖诸如，python，ruby，java等基础环境，建议都装上，然后把各自对应的执行文件都加入到PATH里去吧

理论知识

  1. 用于高亮提示用户编写的代码中存在的不规范和错误的写法，支持 JavaScript、CSS、HTML、Java、PHP、Python、Ruby 等十多种开发语言
  2. 可以理解为sublimelinter是一个代码检测容器，可以在其上边配置数十类语言的监测，但是真正的检测引擎各类语言都有自己对应的引擎

安装步骤

  1. command+shift+p或者windows上ctrl+shift+p调出package control，输入install package

  1. 输入sublimelinter
