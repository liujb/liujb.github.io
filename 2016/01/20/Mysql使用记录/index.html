Mysql使用记录

Posted on 2016-01-20 Edited on 2018-09-03

本文主要记录在Mac OS X平台上使用Mysql的经验和常用命令。

install mysql

  1. brew install mysql
  2. mysql.server start #启动mysql服务
  3. mysql.server stop #关闭mysql服务
  4. 配置文件my.cnf所在位置有可能是/etc/my.cnf或者/usr/local/Cellar/mysql/5.7.10/my.cnf或者/usr/local/etc/my.cnf这三个位置

config your mysql

在终端运行mysql_secure_installation脚本，该脚本会一步步提示你设置一系列安全性相关的参数，包括：设置root密码，关闭匿名访问，不允许root用户远程访问，移除test数据库。当然运行该脚本前记得先启动mysql服务。

这个部分可以参考 Mysql-secure-installtion

basic command

  1. desc tablename; – 查看表结构w
  2. show databases; – 查看所有的数据
  3. show tables; – 查看表
  4. mysql -h localhost -u root -p – 登陆数据库 [-h localhost -P]可选
  5. drop table Recommender_Candidacy_Relation –
  6. show create table department; 显示数据库建表语句
  7. show full fields from department; 显示数据库建表语句
  8. show variables like 'char%'; 显示编码
  9. set names utf8 – 解决了iTerm2终端中mysql中文乱码，我的iTerm2本身是utf8编码，数据库表也是UTF-8编码
