Javascript-data-type

Javascript中基础类型主要有

  • string
  • number
  • bool
  • array
  • object
  • function

其中typeof返回5种类型的字符串

  • string
  • number
  • boolean
  • function
  • object

其中对数组做typeof操作会返回object，所以数组的检测需要额外注意。

var arr = []; console.log(arr instanceof Array)在node环境或者同一global环境下会返回true，但是在跨iframe的情况下是为false，因为在浏览器不同iframe环境下window.Array是不同的。
