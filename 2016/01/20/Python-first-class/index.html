Python-first-class

基本概念

python build

.py .pyc .pyo在python文件中

  1. 其中.py是源程序。
  2. .pyc是编译后形成的文件，对源文件进行编译，就会生成.pyc文件。
  3. .pyo为优化编译后的文件，pyhont -o 源文件就可以将源文件编译为.pyo文件。

python 运行方式

  1. 交互式解释器
  2. 脚本式

python 代码中使用中文

Python中默认的编码格式是 ASCII 格式，在没修改编码格式时无法正确打印汉字，

python3.x默认使用utf8编码方式。

标志符

  1. 和大部分语言一样，都是数字，字符串，下划线，不能以数字开头。
  2. 单下划线开头代表不能直接访问的类属性，需通过类提供的接口进行访问。_hello
  3. 双下划线代表类的私有成员，__foo
  4. 双下划线开头和结尾，代表python里特殊方法的专用标志 如__init__

代码结构

  1. 不使用{}来控制类，函数等其他逻辑。
  2. 多行语句，使用\来表示。
  3. 单行注释#表示，多行'''表示。
  4. 引号分为', ", '''.
  5. 同一行现实多条语句，中间用分号;分割。
  6. 函数之间或类的方法之间用空行分割，表示一段新代码的开始。
