PHP编程基础

Posted on 2016-01-20 Edited on 2018-09-03
基础数据类型
  1. Integer
  2. Float － 也叫Double，用来表示所有的实数
  3. String
  4. Boolean
  5. Array
  6. Object
  7. NULL
  8. resource － 代表外部资源（比如数据库连接），通常他们都为函数的返回值，或者直接传递给函数使用
类型转换
  1. 跟C语言类似，如 $abc = ‘123’; $def = (float)$abc;
  2. 也可以使用内置的方法来进行类型转换。
  3. gettype - 获取数据类型 bool、string、double、null、resource、array、object、unknow type
  4. settype - settype($a, ‘bool’);
可变变量（其实蛮坑爹的）
变量作用域
  1. 内置超级全局变量可以在任何地方使用和可见；
  2. 常量，一旦被声明，可以在全局可见；也就是说可以在函数内部使用；
  3. 在脚本中声明的全局变量在整个脚本中是可见的，但不在函数内部；
  4. 函数内部使用的变量声明为全局变量时，名称要与全局变量保持一致；
  5. 在函数内部创建的变量并声明为静态变量无法在函数外部使用，但是可以在函数的多次执行中保持该值；
  6. 函数内部创建的局部变量，函数执行完之后就被回收了；
操作符
  1. 引用操作符 & – 类似于引用类型
  2. 执行操作符 – 类似于执行mysql语句
  3. 数组操作法 []、+、==、===、!=、!==、<>
  4. 类型操作符 instanceof
获取变量类型
  1. gettype()
  2. is_array()
  3. is_double()、is_float()、is_real()
  4. is_long()、is_int()、is_integet()
  5. is_string()
  6. is_bool()
  7. is_object()
  8. is_resource()
  9. is_null() -
  10. is_scalar() - 是否是一个标量，即布尔值、字符串、整数、浮点数
  11. is_callable() - 是否为函数
  12. is_numeric() - 数字或数字字符串
查看变量状态
  1. isset() - 变量是否存在
  2. unset() - 销毁一个变量
  3. empty() - 变量是否存在，并且变量非空非0
