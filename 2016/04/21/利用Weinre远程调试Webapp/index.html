利用Weinre远程调试Webapp

Posted on 2016-04-21 | Edited on 2020-03-04

本文主要介绍如何利用weinr远程调试webapp。

前置条件

  1. 安装 node、npm 或者 java环境（本文只演示在node环境下的）
  2. 本文是在 OSX 平台进行的，Windows 没有兼容问题。

注意点

  1. 手机和你的电脑必须在同一Wi-Fi。
  2. 绑定的host是本机器（PC）的IP，而不是远程服务器。

Weinre的原理图

其实就是拿你的本机做了个代理，然后通过同步的方式来达到能互相通信。

具体步骤

全局环境下安装weinre npm install -g weinre 如图

执行weinre命令

默认会监听 localhost 的 8080 端口，所以请指定host和port。

例如weinre --boundHost 10.97.27.188 --httpPort 8081 这个IP是本机器的IP（跟你手机Wi-Fi在同一局域网的）不是远程服务器IP地址。

更多用法请参考weinre -h命令
