数据类型

java把所有的数字均认为int类型，所以定义long类型需要在后边添加L

Static

  1. 用于修饰类的属性，此属性就是公共，所有的实例都是该属性
  2. 用于修饰类的方法，直接可以使用class调用
  3. 静态代码块，当类被载入时，静态代码块被执行，且只执行一次。

构造函数声明为private

private的构造函数不能说明此类不能产生实例，产生这个实例的位置一定是在类的内部。
