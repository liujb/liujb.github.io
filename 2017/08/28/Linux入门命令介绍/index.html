Linux入门命令介绍

Posted on 2017-08-28 Edited on 2018-09-01

本文主要记录常用的 Linux 命令，算是学习过程中的沉淀。

基础命令

  1. pwd 查看当前所在目录
  2. whoami, w, who 三个命令查看自己是谁
  3. date 查看当前日期
  4. ls, ls -la 查看当前目录的列表
  5. cd 切换目录
  6. touch fileName 创建文件
  7. mkdir folderName 创建文件夹
  8. rm 移除文件. rm -r 移除文件夹.
  9. cp src dest 复制文件src为dest
  10. mv src dest 移动或者重命名src为dest
  11. scp 远程复制文件。example: scp pay.html xx@10.xx.xx.xx:/home/webroot/app/api/v2/views/webapp

打包解包

tar command

  1. tar cvf filename.tar DIRNAME – 打包
  2. tar xvf filename.tar – 解包（tar是打包不是压缩）
  3. tar zcvf filename.tar.gz DIRNAME – 打包文件夹并压缩为tar.gz
  4. tar zxvf filename.tar.gz – 解压.tar.gz并解包

gzip command

  1. gzip -d filename.gz – 解压文件
  2. gzip filename – 压缩为filename.gz
  3. gunzip gilename.gz – 解压gz文件

unrar命令解压rar文件

需要额外安装软件unrar

  1. unrar e test.rar 解压文件到当前目录
  2. unrar x test.rar /path/to/extract
  3. unrar l test.rar 查看rar中的文件
  4. unrar v test.rar 更详细
  5. unrar t test.rar 测试是否可以成功解压

磁盘相关

  1. du -sh f-weixin-sug – 查看文件夹占用的磁盘空间
  2. du -h – 列出文件夹下所有子文件夹占用磁盘空间的大小
  3. df -h – 查看系统的磁盘使用情况

进阶命令

  1. hostname – 显示主机名
  2. host ip – 通过IP查主机
  3. ping hostname – 通过主机查IP
  4. passwd – 修改密码
  5. top – 显示进程
  6. tail -f /home/xx/php/logs/v2/xx.log.wf | grep wxtransaction -A10 —color
  7. curl
  8. wget
  9. ifconfig en0
  10. ping ifconfig.me
  11. curl ifconfig.me
  12. ps -ef |grep packagename
  13. kill pid
  14. ls -l |grep "^-"|wc -l 查看文件夹下文件个数
  15. ls -lR|grep "^-"|wc -l 查看文件夹下文件个数包括子文件夹
  16. ls -l |grep "^-"|wc -l 查看文件夹下子文件夹的个数

其他技巧

  1. which svn – Find the svn path
  2. ln -s "/Applications/Sublime Text 2.app/Contents/SharedSupport/bin/subl" /usr/local/bin/sublime – 创建软链
  3. alias sublime='open -a "Sublime Text 2"'
  4. /usr/share/ – 这个文件夹下面的东东是干嘛的
  5. iconv -f UTF-16LE -t UTF-8 tmp-dir/a.txt > to/a.txt

给用户增加sudo权限

  1. 用root帐号登录或者su到root。
  2. 增加sudoers文件的写权限： chmod u+w /etc/sudoers
  3. vim /etc/sudoers 找到 root ALL=(ALL) ALL 在这行下边添加 dituhui ALL=(ALL) ALL (ps:dituhui代表是你要添加sudo权限的用户名)
  4. 除去sudoers文件的写权限： chmod u-w /etc/sudoers
