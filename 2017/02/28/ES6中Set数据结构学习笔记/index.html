什么是Set

  • 它类似于数组，但是成员的值都是唯一的。
  • Set函数接受数组（或类似数组的对象）作为参数，用于初始化。
  • 向Set中加入值不会发生类型转换，Set内部判断两个值是否相同的算法类似===（NaN除外）。这意味着两个对象总是不想等的。

操作方法

  1. add(el) – 添加某个值，返回Set结构本身。
  2. delete(el) – 删除某个，返回bool，表示删除成功与否。
  3. has(el) – 表示参数是否为Set的成员，用于判断数组内是否有该值。
  4. clear() – 清除所有成员，没有返回值
